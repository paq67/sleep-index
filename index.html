01 — Context

What is sleep debt?

Sleep debt is the cumulative effect of not getting enough sleep. When you consistently sleep less than your body needs — typically 7–9 hours for adults — the deficit compounds. Unlike financial debt, sleep debt can't simply be "paid back" with a weekend lie-in. Research shows that chronic sleep deprivation fundamentally alters cognitive performance, emotional regulation, metabolic function, and immune response.

The consequences ripple far beyond the individual. Drowsy workers cost the U.S. economy an estimated $411 billion annually in lost productivity. Road accidents linked to fatigue claim thousands of lives. Healthcare systems absorb billions in treating conditions exacerbated by poor sleep. Sleep debt is not just a personal inconvenience — it's a systemic economic drain hiding in plain sight.

04 — Takeaway

Sleep is not optional

The data paints a clear picture: sleep debt is not merely fatigue — it's a compounding liability that degrades cognition, health, and economic output. Addressing this crisis requires systemic change — from workplace policies that respect circadian rhythms to public health campaigns that treat sleep as foundational infrastructure, not luxury. Every hour of sleep reclaimed is an investment in human capital.

import pandas as pd

df = pd.read_csv("/content/drive/MyDrive/data/Sleep_Health_and_Lifestyle_Dataset.csv")
Code [2] 
# Normalize column names safely
df.columns = (
    df.columns
    .str.strip()
    .str.lower()
    .str.replace(" ", "_")
    .str.replace(r"[^\w_]", "", regex=True)
)

print(df.columns.tolist())
Code [3] 
# Ensure column exists
assert "blood_pressure" in df.columns, "Column still not found"

df[["systolic_bp", "diastolic_bp"]] = (
    df["blood_pressure"]
    .str.split("/", expand=True)
    .astype(int)
)
Code [4] 
IDEAL_SLEEP = 7
df["sleep_debt"] = IDEAL_SLEEP - df["sleep_duration"]

df["productivity_score"] = (
    df["daily_steps"] * 0.3
    + df["physical_activity_level"] * 0.4
    + (10 - df["stress_level"]) * 10
)

df["health_risk"] = (
    df["heart_rate"] * 0.3
    + df["systolic_bp"] * 0.2
    + df["stress_level"] * 5
)
